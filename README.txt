MovieAppProject είναι το project που θέλετε να υλοποιήσουμε, το οποίο επεξηγώ στο pdf.
(Υλοποίηση έγινε στο WebStorm 2022 - Ανοιγεται το φακελο MovieApp - κανετε install dependencies
& φτιαχνετε npm configuration με Command: start & Scripts: start)

Επίσης έβαλα το .fig αρχείο που πήρα απο το figma.

Το AppCalculator είναι το calculator που είπατε να κάνουμε στην αρχή του εξαμήνου.



To Repository βρίσκεται στο   https://github.com/KonstBet/MovieAppProject ( το έβαλα public )
